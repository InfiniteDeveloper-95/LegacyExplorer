# Legacy Explorer (Beta)

Универсальная утилита для организации файлов, распаковки старых и проприертарных архивов, конвертации изображений и многих других задач.
Дизайн выполнен в классическом стиле Windows 95/98.

### 🔍 Основные функции:

* **Сортировка:** Быстрое распределение файлов из конкретной папки и подпапок в одну папку и сортировка всех файлов из конкретной папки в несколько тематических папок по типу Pictures, Documents, Archives, 3D Models и другое.
* **Архиватор:** Поддержка распаковки разных игровых, старых и специфических архивов. **Поддерживаемые архивы: Quake `PAK`, Dune II `PAK`, Doom `WAD`, `GRP`, `BRARCHIVE`, `APF`, `VP`, `VPP`, Descent 1 `HOG`, `PBO`, `LECF` и другие современные архивы.**
* **Работа с файлами:** Валидация и форматирование JSON и другое.
* **Конвертирование:** конвертация одного формата файла в другой, но в данной утилите по большей части есть конвертеры картинок. **Поддерживаемые форматы: `JPG`, `WebP`, `BMP`, `PCX`, `TIM`, `TIFF`, `ICO`, `ART`(контейнер с тайлами из Duke Nukem), `TGA` конвертируются В `PNG`, а также `PNG` конвертируется в `WebP` и `JPG`. Также есть создание `ICO` из ваших картинок, то есть вам нужны слои картинок чтобы из них сделать `ICO`, для этого подходят идеально размеры: **16х16, 32х32, 48х48, 64х64, 128х128, 256х256.** И, как вишенка на торте вы можете сжимать `PNG` **без потерь** и **с потерями(если конкретно, то квантование до 256 цветов).**
* **Интерпретатор BASIC:** простой интерпретатор BASIC, здесь он ради забавы, рекомендуется писать код только капсом.
* **DOS Mode:** Режим DOS здесь тоже ради забавы, но он умеет выполнять базовые команды, такие как mem, dir, cd, del и другие. Также некоторые команды я добавляю от себя(которых в оригинальном DOS не было).
* **Свой формат архива:** я добавил `LECF` архивы, использующие **сжатие LZMA**. `LECF` расшифровывается как **Legacy Explorer Custom Format.** Хотя я добавил свой формат архива ради забаавы, он может быть полезным, но тем не менее `.7z` архивы, которые тоже используют **LZMA** более привычны и удобны для повседневных задач.

### 🚀 Как запустить:
1. Перейдите в раздел **Releases** на странице проекта.
2. Скачайте архив с последней версией.
3. Запустите `LegacyExplorer.exe`.
